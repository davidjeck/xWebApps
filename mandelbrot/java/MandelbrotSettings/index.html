Mandelbrot Examples

This page contains a variety of images created with the Java Mandelbrot Viewer program or the Mandelbrot Viewer web app. Click on an image to open the settings file for that image in the web app version of the viewer program. (Note that some of the examples require high precision computations and will take a long time to render. Increasing the number of "Workers" in the program might decrease the computation time.)

Clicking on the word "XML" below the image will show the parameter settings for the image, in XML format. (You don't need to understand XML to use the settings). Right-click "XML" and choose a command such as "Save Link As" to download the correspondings settings into a file. You will then be able to open the file with the Java program to create a larger version of the image that you can then explore further. Just use the "Open Settings File..." command in the "File" menu.

The settings can also be loaded into the JavaScript version of the program, by clicking "Show XML Import/Export" in that program and copy-and-pasting the XML text into the dialog that pops up. See the instructions for that version for more information about how to use the settings.

(This image gallery is just a collection of images that I have saved over the years for one reason or another. I have not organized the gallery or weeded out less interesting examples.)

--David Eck
